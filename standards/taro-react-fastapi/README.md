# Taro React FastAPI Pack

这个 Pack 定义当前项目默认采用的技术栈标准：

- 微信小程序前端
- Taro
- React
- TypeScript
- Tailwind CSS
- Python FastAPI 后端

这是当前项目的默认 Pack 入口文档。

## 这个 Pack 负责什么

这个 Pack 负责定义：

- 小程序与后端的推荐结构
- 前端开发与组件约束
- 后端服务边界
- 编码规范
- 开发工作流
- 验证与脚本约束

## 设计意图

这个 Pack 面向的是：

- 由 Codex 持续参与开发
- 微信小程序作为第一交付形态
- 轻前端 + 可扩展后端
- 后续可能加入 AI 辅助能力

目标不是追求技术炫技，而是保证：

- 容易被 agent 理解
- 易于持续迭代
- 易于验证
- 易于从 V0 升级到 V1
